# bingo_cards
Create UK-style 1-90 bingo cards to print out

## Dependencies

Python libraries

- pycairo
- numpy

Fonts

- Fira Mono
- Fira Sans

## Running the generator

```sh
mkdir output
python generate_pdf.py
```

Each run creates a new PDF file in `output/`.  Each page in the PDF has 2 strips of 6 cards.
